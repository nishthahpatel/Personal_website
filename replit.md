# vCard - Personal Portfolio

## Overview
A fully responsive personal portfolio website built with HTML, CSS, and JavaScript. This is a static website showcasing a personal portfolio with sections for About, Resume, Portfolio, Blog, and Contact.

## Project Structure
- `index.html` - Main HTML file with all page content
- `assets/css/style.css` - All styling for the website
- `assets/js/script.js` - Interactive functionality (navigation, modals, filters)
- `assets/images/` - All images and icons
- `server.js` - Simple Node.js HTTP server to serve static files on port 5000

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Node.js (for serving static files in Replit)

## Setup
This project uses a simple Node.js HTTP server to serve static files. The server is configured to:
- Run on port 5000 (required for Replit webview)
- Bind to 0.0.0.0 (required for Replit proxy)
- Disable caching to ensure updates are visible
